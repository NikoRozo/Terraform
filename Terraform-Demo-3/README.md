# Terraform
Uso de Outputs y recursos de Security Group

## Para ejecutar cualquier proyecto ejecute:
```bash
# Descarga los plugin del proveedor
terraform init
# Validar sintaxis
terraform validate
# Plan
terraform plan
# Ejecutar
terraform apply
# Ejecutar - omitir aprobaci√≥n
terraform apply -auto-approve
# Destruir Infraestructura
terraform destroy
```
